# 芝麻史记 - 宝宝排便记录应用

## 项目介绍

芝麻史记是一款专为记录宝宝排便情况设计的网页应用。通过简洁直观的界面，家长可以轻松记录宝宝每次排便的详细信息，包括形状、颜色、气味、时长和心情等，并通过数据分析了解宝宝的排便健康状况。

## 功能特点

- **美观的加载动画**：首次进入应用时展示精美的加载动画和随机名言
- **详细的记录表单**：记录排便的各项指标，包括日期、时间、形状、颜色、气味、时长和心情
- **专业的健康提示**：针对不同的排便形状、颜色和气味提供专业的健康提示
- **直观的历史记录**：以卡片形式展示历史记录，支持搜索和按心情筛选
- **丰富的数据分析**：通过多种图表展示排便数据分析，包括形状分布、颜色分布、气味分布、心情趋势、每日次数和时长趋势
- **数据导出功能**：支持将记录导出为CSV或JSON格式
- **深色模式支持**：自动适应系统的深色模式设置
- **响应式设计**：适配各种屏幕尺寸，包括手机、平板和电脑

## 使用方法

1. **访问应用**：打开index.html文件，等待加载动画结束后自动跳转到记录页面
2. **记录排便**：在左侧表单中填写排便信息，包括日期、时间、形状、颜色、气味、时长和心情，点击"保存记录"按钮保存
3. **查看历史**：在右侧历史记录区域查看所有记录，可以通过搜索框搜索或通过心情下拉框筛选
4. **分析数据**：在数据分析区域查看各种图表，了解宝宝的排便健康状况
5. **导出数据**：点击导出按钮，选择导出格式（CSV或JSON）

## 技术栈

- HTML5 + CSS3 + JavaScript
- TailwindCSS：用于快速构建响应式界面
- Chart.js：用于数据可视化
- Tippy.js：用于提示框
- Font Awesome：用于图标
- Google Fonts：用于字体

## 注意事项

- 所有数据均存储在浏览器的本地存储中，清除浏览器缓存会导致数据丢失
- 建议定期导出数据以备份
- 本应用提供的健康提示仅供参考，如有健康问题请咨询医生

## 自定义

- 可以在css/global.css文件中修改颜色变量，自定义应用的主题色
- 可以在js/global.js文件中修改一言API的替换词，自定义加载页面的名言内容
